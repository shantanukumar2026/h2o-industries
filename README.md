## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## About H2 Industries

H2 Industries is an engineering and manufacturing company focused on industrial water-management solutions. With over a decade of experience delivering robust sampling stations, modular enclosures, and precision drainage systems, we design products that protect waterways and reduce environmental impact while meeting strict industrial standards.

Industry context: clean energy and low-emission technologies are reshaping heavy industry and infrastructure. Demand for solutions that integrate with evolving hydrogen production, electrolyser installations, and resilient coastal logistics is growing — H2 Industries builds products to be compatible with these trends and future-proof operations.
